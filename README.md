## Overview

Sistem ini adalah website yang digunakan untuk memudahkan pengguna yang ingin melakukan peminjaman/rental mobil listrik dan memudahkan mitra untuk menyewakan mobil listrik yang dimiliki.

Aktor utama pada sistem ini adalah pengguna, mitra dan admin. Dimana pengguna dapat menyewa/rental mobil listrik, mitra dapat memasarkan mobil listrik dan admin dapat memanajemen sistem.
